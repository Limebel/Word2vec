# Word2vec
Implementation of Word2vec skip-gram with negative sampling model trained on wikitext database  
Model is implemented and described well inside word2vec_vfull.ipynb file, feel free to explore it  
Link to data origin: https://huggingface.co/datasets/Salesforce/wikitext
